# Naija Lightning Wallet (NLW)

**A lightweight, non-custodial Bitcoin wallet built for low-bandwidth environments.**

---

## Overview
Naija Lightning Wallet (NLW) is an open-source Bitcoin wallet designed for regions with:
- Unstable internet
- Expensive data
- Low-end Android devices

It combines Bitcoin on-chain transactions and Lightning Network payments into a simple, fast, and reliable mobile experience.

---

## Mission
Make Bitcoin usable as everyday money in emerging markets.

---

## Features (In Progress)

### Lightning + On-chain
- Send and receive Bitcoin
- Lightning payments (planned)
- Automatic fallback to on-chain

### Low-Bandwidth Optimization
- Compact block filters (BIP157/158)
- Minimal data usage
- Works on slow networks

### Non-Custodial
- Full control of private keys
- 12-word seed phrase backup

### Simple UX
- Easy “Send” and “Receive”
- Local currency display (₦)
- Beginner-friendly design

---

## Why This Matters
Most Bitcoin wallets are built for users with:
- Stable internet
- High-end devices
- Technical knowledge

NLW focuses on the opposite:
Making Bitcoin work in real-world conditions like Nigeria and other emerging markets.
